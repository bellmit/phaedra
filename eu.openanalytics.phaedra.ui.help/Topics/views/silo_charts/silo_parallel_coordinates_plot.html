Silo Parallel Coordinates Plot

The Parallel Coordinates Plot is a common way of visualizing high-dimensional geometry and analyzing multivariate data. Special for this plot is that you select more than one feature on the x axis.

This plot can be used to see the relation between different cells over a series of features. You can use features from different silo groups.


To this chart, you can only add more parallel coordinates layers. You can group them differently or add a different aggregation to them.

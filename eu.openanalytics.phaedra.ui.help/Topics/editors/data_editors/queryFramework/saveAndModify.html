Saving and modifying queries

Saving a Query

You can control your queries using the action buttons in the top right corner:


 Save a query (overwrite the previous version)

Save As, save a query with a new name

Delete a query

Execute a query

If you constructed a new query (from scratch):


  • Press the SAVE (left most) button
  • The saved query appears in the Navigator under the specified Category
  • You will be the owner of this query

If you opened and modified an existing query:


  • Even if you modify the name of the query, Using “Save” will OVERWRITE the existing query !! (the name will simply change in the navigator)
  • If you only changed the List Category (Public, Example, Private), the query will be moved to the new category
  • So if you like to keep the previous query as well, use the "SAVE AS" button.
Modifying a Query

Private Queries (“MyQueries”):

You have constructed these queries, so you are the owner of these queries. You are the only one that sees these queries in the navigator and you are the only one that is able to modify or delete these queries.

Public / Example Queries:

You can only modify or delete queries that you own.

Note: If you are not allowed to modify or delete, the corresponding buttons will be greyed out.

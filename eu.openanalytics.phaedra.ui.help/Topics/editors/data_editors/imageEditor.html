ROI Editor

The ROI Editor in Phaedra is actually an Image Region/Cells Creator, not a general image editor.

The ROI editor allows to indicate regions of interest in the images (ROIs). You can configure these regions by adding a classification well feature in the Protocol. Each "class" will be used as a region. The ROIS have a color as configured in the protocol. The example below shows a mouse whole-brain slice (35,000 by 25,000 pixels) with amyloid plaques detected.


The user has to draw the different brain regions as indicated in the top right panel, as follows:


  1. Click on the region you want to draw, e.g. Hippocampus
  2. Draw the region with the mouse.
  3. If you made a mistake, click "Undo"
  4. Do the same for the other regions.
  5. Click Save to store the regions

The regions are stored as image overlay channels. A number of features can now be calculated. These have been entered in the protocol class as calculated features. In this example, Phaedra will calculate the number and the area of the plaques within each region, and stores the values in the Database.

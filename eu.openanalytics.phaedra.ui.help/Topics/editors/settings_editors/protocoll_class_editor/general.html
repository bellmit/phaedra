General Settings

This section contains the general settings and permissions of the protocol class.

General


Setting Description
Name The name of the protocol class
Id The unique identifier of the protocol class (not editable)
Description A description of the protocol class
Lock Status If locked, the protocol class is frozen. Only administrators can unlock the protocol class.
Development Status If in development, all team members can make changes to the protocol class.
Default Link Source The default plate definition system to use when linking plate definitions.
Default Well Feature The default well feature that will be selected in the Feature Selector.
Default Importer The default import wizard to use when importing new plates.
Low Control Type The default Low Control type for all well features.
High Control Type The default High Control type for all well features.

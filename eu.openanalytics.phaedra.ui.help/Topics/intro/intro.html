Introduction

About this document

This document explains the various components of Phaedra, and how to use them. Since Phaedra is a large application with many specialized functions, not all sections of this document may apply to you. Instead, it may be more appropriate to use this document as a hands-on guide, to be consulted during the usage of Phaedra.

Getting help in Phaedra

At any time, you can press the F1 key in Phaedra to open a Help screen. This screen will attempt to display help on the view currently displayed in Phaedra. This is called context-sensitive help. If Phaedra is unable to show context-sensitive help, it will display an index of help topics instead.

Getting support for Phaedra

If you want to request access for Phaedra, or you encounter a problem in Phaedra the Help system does not have a solution for, please contact the Phaedra support team.

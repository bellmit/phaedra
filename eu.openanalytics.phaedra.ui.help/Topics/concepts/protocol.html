Protocol

The protocol defines how the assay is performed. It refers to a protocol class, which is a blueprint for the data that will be loaded into the protocol.

The protocol class defines:


  • A list of well features. These are measurements, aggregated to a single value per well. A high-content assay typically contains a long list of well features.
  • A list of subwell features. These are measurements on the cell or sub-cell level. They are usually obtained directly from the instrument or image analysis results.
  • A list of image channels. Each channel corresponds to a channel or mode used by the instrument, for example Brightfield or Alamar Blue Fluorescence.

Refer to the description of Well Feature and Subwell Feature for more information.
